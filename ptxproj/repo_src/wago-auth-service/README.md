# Auth Service

## Static code analysis

For the JavaScript and CSS files use ESLint to run checks. You will need NodeJS to do so.  

```bash
npm install --global eslint
cd www/auth
cd www/auth
npx eslint
```

To run ESLint inside VSCode, install Microsoft's [ESLint extenstion](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) and follow its instructions for setup.
