# Introduction to WAGO Daemon Utils

This package consists of four libraries:

## WAGO Logging

The purpose of this library is to provide an easy way to setup 
logging.

## WAGO Opt-Parsing

The purpose of this library is to provide an easy way to parse
program arguments.

## WAGO Privileges

The purpose of this library is to provide an easy way to drop 
privileges.

## WAGO Trace

The purpose of this library is to provide possibility to trace
taken branches and their execution time on program runtime.
Also interaction with other programs are visible with the help
of the kernel tracer.
